ファイルの保存

ファイルの保存

ファイルは独自バイナリ形式(ied形式/iedx形式)とXML形式で保存可能です。XML形式の場合は、フォント属性などが保存されませんので、ご注意ください。XMLのタグの定義などに関しては、プログラム付属のiEditDoc.iedを参照してください。
